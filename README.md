# Astro - Your Photos, Your Universe

Astro is a beautiful, privacy-focused iOS client for [Immich](https://immich.app). Rediscover your memories with a native, gesture-driven experience.

## Features

- **Native Performance**: Built with Swift for a buttery smooth experience on iOS.
- **Privacy First**: Your photos stay on your server. No third-party tracking.
- **Beautiful Design**: A modern, gesture-driven interface designed for photos.
- **Memories**: Rediscover your past with intelligent "On This Day" collections.

## Tech Stack

This landing page is built with:
- **Framework**: [Next.js 15](https://nextjs.org/) (Static Export)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Motion](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)

## Deployment

This site is configured for **Cloudflare Pages** using static export (`output: 'export'`).

1.  Connect your repository to Cloudflare Pages.
2.  Set the **Build command** to `npm run build`.
3.  Set the **Build output directory** to `out`.
